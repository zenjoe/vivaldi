### Get started overview

Clone

```
git clone https://github.com/zenjoe/vivaldi.git
cd vivaldi
```

Install dependencies

```
npm install
```


Start a webpack development server running at localhost:3000
```
npm start
```

_Specify port on dev server_ `PORT=1337 npm start`
